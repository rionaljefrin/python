# Student Marks Analysis (Pandas + NumPy + Matplotlib)

This project reads a CSV file of 30 students and their marks in 3 subjects and performs basic data analysis and visualization.

## What this script does

- Reads CSV using Pandas
- Shows quick look and summary statistics
- Finds top 5 students by total marks
- Computes subject-wise mean, min, max using NumPy
- Plots average marks vs subject using Matplotlib

## How to run

### 1. Install requirements

pip install -r requirements.txt

### 2. Run the script

python analysis.py

## Output

- Console summary of data
- Bar chart of average marks per subject
